# Student Marks & Grade Calculator

## Description
A beginner Python project that calculates total marks,
average, grade, and pass/fail status of a student.

## Features
- Accepts student name
- Takes marks for multiple subjects
- Calculates total & average
- Calculates percentage
- Determines grade and pass/fail
- Validates marks between 0 and 100

## Concepts Used
- Python basics
- Loops
- Lists
- Conditional statements
- Validation

## How to Run
Run the Python file using any Python interpreter or Pydroid 3 on mobile.
